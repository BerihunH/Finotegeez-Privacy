# Privacy Policy - Finote Geez Learning

## 1. Information We Collect
- The app processes **voice input** through the Google Speech-to-Text API (Amharic) to provide pronunciation feedback. **No voice data is stored permanently.**  
- **Wordwall integration** uses the user’s **first name** to personalize interactive exercises.  
- All other app features work **offline** and do not collect personal information.

## 2. How We Use Information
- Voice input is used **only for pronunciation guidance**.  
- First name data is sent to Wordwall **solely to display and personalize quizzes**.  
- Offline app features **do not collect or transmit any data**.

## 3. Third-Party Services
- **Google Speech-to-Text API** – Converts spoken Amharic words into text for pronunciation exercises.  
- **Wordwall** – Provides interactive quizzes using first names only.  
- **Google Play Services** – Used for app updates and compatibility.

## 4. Data Sharing
- Voice and name data are **not shared with any other third parties** beyond what is required for Google Speech-to-Text or Wordwall functionality.

## 5. User Rights
- Users can delete their app data by **uninstalling the app**.

## 6. Security
- All transmitted data is handled **securely** according to industry-standard practices.

## 7. Contact
- For questions about privacy or data usage, contact: **finotegeez@gmail.com**
